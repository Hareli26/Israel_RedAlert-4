# 🔴 Red Alert Monitor v4.0 — התרעות צבע אדום

מערכת ניטור התרעות פיקוד העורף בזמן אמת לשולחן העבודה של Windows.
שואב מידע מ-API הרשמי של פיקוד העורף (OREF) ומציג התרעות בממשק גרפי מלא.

---

## ✨ תכונות עיקריות

### 🚨 התרעות בזמן אמת
- שאילתה לשרת פיקוד העורף כל **2 שניות**
- **סינון לפי ישוב** — קבלת התרעות רק לאזור שלך
- סינון אוטומטי של התרעות בדיקה (`cat=101`)
- תמיכה בסוגי אירועים: טיל / כלי טיס / אירוע חירום

### 🖥️ ממשק משתמש

| רכיב | תיאור |
|------|--------|
| **FloatingWidget** | ווידג'ט צף בצד המסך — תמיד גלוי, ניתן לגרור |
| **FullScreen Banner** | מסך מלא עם שם הישוב, פרטי האירוע, שעת ההתרעה ושעון חי |
| **MapWindow** | מפת Leaflet מוטמעת עם סמנים לישובים תחת התרעה |
| **ShelterMiniBanner** | רצועה צפה אדומה — עוברת לירוק בשחרור פיקוד העורף |
| **System Tray** | אייקון במגש המערכת עם תפריט מהיר |

### 🗺️ מפה
- נפתחת אוטומטית בקבלת התרעה
- **נסגרת אוטומטית לאחר 10 שניות**
- מסמנת את הישובים שלך + ישובים תחת התרעה פעילה
- תמיכה בשיתוף מיקום Google

### 🏠 זרימת מרחב מוגן
1. קבלת התרעה → FullScreen עם כפתורי **"הבנתי — סגור"** ו-**"אני במרחב המוגן"**
2. לחיצה על "במרחב מוגן" → נסגר FullScreen → מופיע **ShelterMiniBanner** אדום
3. שחרור ע"י פיקוד העורף → הבאנר עובר לירוק: **"ניתן לצאת — פיקוד העורף שחרר"**
   ונסגר אוטומטית לאחר 8 שניות

### 📍 זיהוי מיקום אוטומטי
- בהפעלה ראשונה (ללא ישובים מוגדרים): מזהה מיקום לפי כתובת IP
- מציג דיאלוג עם הישוב שזוהה — ניתן לאשר, לדחות, או לפתוח הגדרות

### 🔊 סאונד
- מספר סוגי אזעקות לבחירה: `standard` / `classic` / `digital` / `silent`
- ניתן להפעיל/לכבות בחלון ההגדרות

## 🚀 הפעלה

### אפשרות א׳ — קובץ BAT (מומלץ)
לחץ פעמיים על:
```
הפעל.bat
```
הסקריפט מתקין תלויות אוטומטית ומפעיל את האפליקציה.

### אפשרות ב׳ — Python ישיר
```bash
pip install -r requirements.txt
python red_alert.py
```

### אפשרות ג׳ — EXE מוכן (לאחר Build)
```
לחץ פעמיים על:  RedAlertMonitor.exe
```

## 🔨 בניית EXE

```bash
build_exe.bat
```

הקובץ `RedAlertMonitor.exe` ייוצר בתיקיית הפרויקט.

**שלבי הבנייה:**
1. התקנת PyInstaller ותלויות
2. בנייה כ-EXE יחיד (`--onefile --windowed`)
3. העתקה לתיקיית הפרויקט
4. ניקוי קבצי build זמניים

## 🔌 API

האפליקציה שואבת מ-API הרשמי של פיקוד העורף:

```
GET https://www.oref.org.il/WarningMessages/alert/alerts.json
```

| תגובה | משמעות |
|-------|---------|
| גוף ריק (BOM בלבד) | אין התרעה פעילה |
| JSON עם `id`, `cat`, `title`, `data` | התרעה פעילה |
| `cat = "101"` | התרעת בדיקה — מסוננת אוטומטית |

## 🛠️ פתרון בעיות

### האפליקציה קורסת ללא הודעת שגיאה
קריסת C++ של Chromium בהרצה כ-Administrator עם GPU מופעל.
**פתרון:** מטופל אוטומטית בקוד — הדגלים הבאים מוגדרים לפני `QApplication`:
```
QTWEBENGINE_CHROMIUM_FLAGS = --disable-gpu --disable-software-rasterizer --no-sandbox
QTWEBENGINE_DISABLE_SANDBOX = 1
```

### מקבל התרעות מכל הארץ
ללא ישוב מוגדר — מתקבלות **כל** ההתרעות בישראל.
**פתרון:** פתח **הגדרות** ← הוסף את הישוב שלך.

### לא מגיע סאונד
בדוק שהסאונד לא מוגדר כ-`שקט` בהגדרות ושהרמקולים מחוברים.

### קבצי לוג לאיתור שגיאות
```
%APPDATA%\RedAlert\crash.log        ← שגיאות Python
%APPDATA%\RedAlert\crash.log.flt    ← קריסות C++ (faulthandler)
```
