# 🔴 מערכת התרעות צבע אדום — Red Alert Monitor v2.0

תוכנת שולחן עבודה לקבלת התרעות פיקוד העורף בזמן אמת.

---

## ✅ דרישות מערכת
- **Windows 10/11**
- **Python 3.8+** — הורד מ: https://www.python.org/downloads/

---

## 🚀 הפעלה

פשוט לחץ פעמיים על:
```
הפעל.bat
```

הסקריפט יתקין אוטומטית את כל הדרישות ויפעיל את התוכנה.

**או ידנית:**
```
pip install PyQt5 requests
python red_alert.py
```

---

## 🖥 תכונות

### חלון צף (Mini Widget)
- **עמדה קבועה** בצד ימין של המסך — ניתן לגרור לכל מקום
- **פועם באדום** בזמן התרעה פעילה
- **מציג** שם ההתרעה, ישובים מוזהרים, ושעה
- **אייקון** מתאים לסוג האיום
- לחיצה כפולה → פותח מסך מלא

### מסך מלא
- **באנר** עם צבע וסוג ההתרעה
- **אנימציה** של האייקון (טיל קופץ, מטוס עף...)
- **דגל המדינה** ממנה יצא האיום (כשידוע)
- **רשת הישובים** המוזהרים
- **היסטוריה** של התרעות אחרונות
- סגירה: ESC / לחיצה כפולה

### סוגי איומים ואייקונים

| קטגוריה | אייקון | תיאור |
|---------|--------|--------|
| ירי רקטות | 🚀 | rocket — אנימציית קפיצה |
| כלי טייס עוין | ✈️ | מטוס — אנימציית טיסה |
| רעידת אדמה | 🌍 | כדור הארץ — רעידה |
| חומרים מסוכנים | ☢️ | קרינה — פעימה |
| חדירת מחבלים | ⚠️ | אזהרה — קפיצה |
| צונאמי | 🌊 | גל — גלישה |
| בדיקה | 🔔 | פעמון |

### מדינת מקור (ניחוש לפי ישוב)
- דרום → 🇵🇸 עזה
- צפון → 🇱🇧 לבנון / חיזבאללה
- מזרח → 🇮🇷 איראן / חות'ים

## ⚙ הגדרות
לחץ על **⚙** בחלון הצף:
- **בחירת ישובים** — לסנן התרעות רק לאזורך
- **צליל** — כיבוי/הדלקה
- **מסך מלא אוטומטי** — פתיחת מסך מלא בהתרעה
- **התרעת בדיקה** — לוודא שהכל עובד

## 🔌 API
- **מקור:** פיקוד העורף הרשמי — `oref.org.il`
